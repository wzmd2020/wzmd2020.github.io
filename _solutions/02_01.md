Wyliczamy \(n\):
\[n = k - \lambda = k - 1\]
Konfiguracja jest symetryczna, więc mamy
\[r = k = n + 1\]
Podstawiamy do wzoru:
\[r(k-1) = \lambda(v-1)\]
\[(n+1)n = 1 \cdot (v - 1)\]
\[v = n^2 + n + 1\]
Otrzymaliśmy \((n^2 + n + 1, n + 1, 1)\)-konfigurację,
<br>
a więc płaszczyznę rzutową rzędu \(n\)
